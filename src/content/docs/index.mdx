## Features

- **Flexible Architecture**: Design networks with any number of layers and neurons
- **Built-in Optimizers**: SGD, Adam, RMSProp, and more for efficient training
- **Activation Functions**: ReLU, Sigmoid, Tanh, Softmax, and others
- **Serialization**: Save and load trained models
- **Regularization**: Dropout, L1/L2 regularization to prevent overfitting
- **GPU Acceleration**: Optional CUDA support for faster training
